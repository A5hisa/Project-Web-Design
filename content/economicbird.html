นกนางแอ่น

ชื่อวิทยาศาสตร์ : Hirundo rustica
วงศ์ : Hirundinidae
อันดับ : Passeriformes

ลักษณะ : นกนางแอ่นเป็นนกขนาดเล็ก ลำตัวยาวประมาณ 15-20 เซนติเมตร รวมหางแฉก ปีกเรียวยาวและหางแฉกเป็นสองแฉกเด่นชัด ขนหางคู่นอกยาวที่สุด สีขนโดยทั่วไปเป็นสีน้ำเงินเข้มหรือดำเหลือบเงา หน้าผากและคอมีสีแดงน้ำตาล มีแถบดำพาดอก หน้าอกและท้องสีขาวหรือครีม นกชนิดนี้มีความคล่องแคล่วในการบิน สามารถโฉบจับแมลงกลางอากาศได้อย่างรวดเร็ว ตัวผู้และตัวเมียมีลักษณะคล้ายกัน


ถิ่นที่อยู่ : นกนางแอ่นบ้านเป็นนกประจำถิ่นและนกอพยพในประเทศไทย พบได้ทั่วไปในทุกภูมิภาค มักอาศัยอยู่ใกล้ที่อยู่อาศัยของมนุษย์ เช่น โรงนา อาคาร หรือใต้หลังคา รวมถึงพื้นที่โล่ง ทุ่งนา และริมแม่น้ำ


การสืบพันธุ์ : นกนางแอ่นบ้านจะสร้างรังด้วยโคลนผสมเศษหญ้าและวัสดุอื่น ๆ ทำเป็นรูปถ้วยเปิดด้านบน มักทำรังใต้ชายคาหรือสิ่งก่อสร้าง วางไข่ครั้งละ 3-5 ฟอง ใช้เวลาฟักประมาณ 14 วัน โดยทั้งพ่อและแม่นกช่วยกันเลี้ยงลูกอ่อน

นกกรงหัวจุก (นกปรอดหัวโขน)

ชื่อวิทยาศาสตร์ : Pycnonotus jocosus
วงศ์ : Pycnonotidae
อันดับ : Passeriformes

ลักษณะ : นกกรงหัวจุกเป็นนกขนาดกลาง ยาวประมาณ 20 เซนติเมตร ลำตัวส่วนบนมีขนสีน้ำตาลแกมดำ ส่วนคอและท้องสีขาว หัวสีดำมีขนจุกตั้งขึ้นเด่นชัด มีแถบสีขาวอยู่ใต้คาง และมีจุดเด่นคือมีเส้นสีแดงสดอยู่หลังตาคล้ายหนวด และมีขนคลุมก้นสีแดง หางยาวปลายหางมีสีขาว เสียงร้องไพเราะและหลากหลาย เป็นนกที่ได้รับความนิยมเลี้ยงไว้ร้องในกรง


ถิ่นที่อยู่ : พบได้ทั่วไปในทุกภูมิภาคของประเทศไทย เป็นนกประจำถิ่น มักอาศัยอยู่ตามสวนผลไม้ ป่าละเมาะ พื้นที่โล่ง และเขตที่ใกล้กับชุมชนมนุษย์


การสืบพันธุ์ : นกกรงหัวจุกทำรังเป็นรูปถ้วยอย่างหลวม ๆ ด้วยกิ่งไม้และเศษหญ้า ตามพุ่มไม้หรือง่ามไม้ที่สูงจากพื้นดินประมาณ 1-3 เมตร วางไข่ครั้งละ 2-4 ฟอง ใช้เวลาฟักประมาณ 12-14 วัน พ่อและแม่นกช่วยกันเลี้ยงลูกอ่อน

นกเขาชวา

ชื่อวิทยาศาสตร์ : Geopelia striata
วงศ์ : Columbidae
อันดับ : Columbiformes

ลักษณะ : นกเขาชวาเป็นนกขนาดเล็กถึงกลาง ลำตัวเรียวยาว มีความยาวประมาณ 20-22 เซนติเมตร หางแหลมยาว ลำตัวมีสีน้ำตาลแกมเทา จุดเด่นคือมีลายเส้นสีดำสลับสีเทาพาดขวางอย่างชัดเจนบริเวณคอและด้านข้างลำตัว หน้าอกมีสีชมพูอมเทา เสียงร้องไพเราะและมีจังหวะเป็นเอกลักษณ์ (อุด-อุอุอุอู้) จึงเป็นที่นิยมเลี้ยงไว้ฟังเสียงและแข่งขัน


ถิ่นที่อยู่ : เป็นนกประจำถิ่นที่พบได้ทั่วไปในทุกภูมิภาคของประเทศไทย ถูกนำเข้ามาจากอินโดนีเซียและแพร่กระจายพันธุ์ในภายหลัง ชอบอยู่ในพื้นที่โล่ง ทุ่งนา สวนผลไม้ ป่าละเมาะชายทุ่ง และบริเวณใกล้ชุมชน


การสืบพันธุ์ : นกเขาชวาทำรังแบบง่าย ๆ ด้วยกิ่งไม้แห้งและหญ้า วางรังตามง่ามต้นไม้ พุ่มไม้ หรือตามสิ่งก่อสร้าง วางไข่ครั้งละ 2 ฟอง ใช้เวลาฟักประมาณ 11-14 วัน โดยทั้งพ่อและแม่นกจะช่วยกันสร้างรัง ฟักไข่ และเลี้ยงลูกอ่อน

นกค็อกคาเทล

ชื่อวิทยาศาสตร์ : Nymphicus hollandicus
วงศ์ : Cacatuidae
อันดับ : Psittaciformes

ลักษณะ : นกค็อกคาเทลเป็นนกแก้วขนาดเล็กถึงกลาง ลำตัวยาวประมาณ 30-33 เซนติเมตร ขนสีเทา สีดั้งเดิม หน้าสีเหลือง และแก้มมีจุดสีส้มแดงเด่นชัด ในตัวผู้จะสดกว่าตัวเมีย มีขนจุกบนหัวที่สามารถขยับขึ้นลงได้ตามอารมณ์ นิสัยอ่อนโยน เชื่อง และสามารถฝึกผิวปาก เลียนแบบเสียง และเลียนเสียงพูดได้ (โดยมีความสามารถในการเลียนเสียงผิวปากเป็นที่โดดเด่น)


ถิ่นที่อยู่ : มีถิ่นกำเนิดในประเทศออสเตรเลีย มักอาศัยอยู่ในทุ่งหญ้า ป่าโปร่ง หรือใกล้แหล่งน้ำ โดยส่วนใหญ่มักอยู่ร่วมกันเป็นกลุ่มเล็ก ๆ หรือเป็นคู่


การสืบพันธุ์ : นกค็อกคาเทลจะผสมพันธุ์ในฤดูฝน (ซึ่งถูกกระตุ้นโดยปริมาณน้ำฝน) ทำรังในโพรงไม้ (มักเป็นโพรงต้นยูคาลิปตัส) วางไข่ครั้งละ 4-7 ฟอง ใช้เวลาฟักประมาณ 17-23 วัน โดยทั้งพ่อและแม่นกช่วยกันดูแลและผลัดกันกกไข่

นกซันคอนัวร์

ชื่อวิทยาศาสตร์ : Aratinga solstitialis
วงศ์ : Psittacidae
อันดับ : Psittaciformes

ลักษณะ : นกซันคอนัวร์เป็นนกแก้วขนาดกลาง ลำตัวยาวประมาณ 30 เซนติเมตร รวมหาง มีสีสันสดใสสวยงาม ลำตัวสีเหลืองทอง หน้าสีส้ม ปีกและหางมีสีเขียวและน้ำเงิน (นกวัยเยาว์จะมีสีเขียวเด่นกว่าและสีสดใสจะปรากฏชัดเมื่ออายุประมาณ 1.5-2 ปี) นิสัยร่าเริง ฉลาดสูง และขี้อ้อนมาก แต่เป็นนกที่ขึ้นชื่อเรื่องเสียงร้องที่ดังและแหลมมาก


ถิ่นที่อยู่ : มีถิ่นกำเนิดในอเมริกาใต้ บริเวณทางตะวันออกเฉียงเหนือของทวีป โดยเฉพาะประเทศบราซิล กายอานา และซูรินาม อาศัยอยู่ในป่าเขตร้อน ป่าโปร่ง และป่าซาวันนาใกล้แหล่งน้ำ


การสืบพันธุ์ : นกซันคอนัวร์ทำรังในโพรงไม้ วางไข่ครั้งละ 3-5 ฟอง ใช้เวลาฟักประมาณ 23-27 วัน โดยแม่นกจะทำหน้าที่ฟักไข่เป็นหลัก และพ่อนกจะช่วยหาอาหารและดูแลรัง

นกฟอพัส

ชื่อวิทยาศาสตร์ : Forpus coelestis
วงศ์ : Psittacidae
อันดับ : Psittaciformes

ลักษณะ : นกฟอพัสเป็นนกแก้วขนาดเล็กที่สุดในโลก (Parrotlet) ลำตัวยาวประมาณ 12-13 เซนติเมตร สีขนหลักเป็นสีเขียวหรือฟ้าสด ตัวผู้จะมีแถบสีน้ำเงินเข้มเด่นชัดที่ปีกและตะโพก ในขณะที่ตัวเมียจะมีสีเขียว หรือสีพื้นฐาน ทั้งตัว เป็นนกที่เชื่อง ฉลาด และชอบอยู่เป็นคู่ แต่มีพฤติกรรมหวงอาณาเขตสูงเมื่อเทียบกับขนาดตัว


ถิ่นที่อยู่ : มีถิ่นกำเนิดในทวีปอเมริกาใต้ โดยเฉพาะในประเทศเปรูและเอกวาดอร์ (บริเวณฝั่งตะวันตกของเทือกเขาแอนดีส) ชอบอยู่ในป่าละเมาะ พื้นที่กึ่งแห้งแล้ง และพื้นที่โล่ง


การสืบพันธุ์ : ทำรังในโพรงไม้ วางไข่ครั้งละ 4-8 ฟอง ใช้เวลาฟักประมาณ 18-20 วัน โดยแม่นกเป็นผู้ฟักไข่ และพ่อนกคอยหาอาหารมาเลี้ยงทั้งแม่นกและลูกอ่อน

นกเลิฟเบิร์ด

ชื่อวิทยาศาสตร์ : Agapornis roseicollis
วงศ์ : Psittaculidae
อันดับ : Psittaciformes

ลักษณะ : นกเลิฟเบิร์ดเป็นนกแก้วขนาดเล็ก ลำตัวยาวประมาณ 15-17 เซนติเมตร สีดั้งเดิมคือลำตัวสีเขียว มีใบหน้าและอกส่วนบนสีชมพูอมส้ม และมีตะโพกสีฟ้าสด นกที่เลี้ยงมีสีหลากหลายจากการกลายพันธุ์ เป็นนกที่ชอบอยู่เป็นคู่ และมีนิสัยรักคู่มาก ตัวผู้และตัวเมียมีสีสันคล้ายกันมาก ไม่มีความแตกต่างทางเพศที่ชัดเจน และมักรวมตัวกันเป็นฝูงเล็ก ๆ ในธรรมชาติ


ถิ่นที่อยู่ :มีถิ่นกำเนิดในทวีปแอฟริกาตะวันตกเฉียงใต้ โดยเฉพาะในประเทศนามิเบียและแองโกลา พบได้ในพื้นที่กึ่งแห้งแล้ง ป่าโปร่ง ทุ่งหญ้า และตามริมทะเลทราย โดยมักอาศัยอยู่ใกล้แหล่งน้ำ


การสืบพันธุ์ : นกเลิฟเบิร์ดทำรังในโพรงไม้หรือตามรังนกอื่นที่ถูกทิ้งร้าง โดยแม่นกจะใช้ปากคาบวัสดุทำรัง (เช่น เปลือกไม้หรือใบหญ้า) ไปสอดไว้ตามขนเพื่อขนย้าย วางไข่ครั้งละ 4-7 ฟอง ใช้เวลาฟักประมาณ 20-27 วัน โดยแม่นกฟักไข่ ส่วนพ่อนกหาอาหารมาเลี้ยง

นกแก้วมาคอร์ฟ้า-เหลือง

ชื่อวิทยาศาสตร์ : Ara ararauna
วงศ์ : Psittacidae
อันดับ : Psittaciformes

ลักษณะ : นกแก้วมาคอร์ฟ้า-เหลืองเป็นนกแก้วขนาดใหญ่ ลำตัวยาวประมาณ 76-86 เซนติเมตร มีสีสันสดใส ได้แก่ สีน้ำเงินสดบนหลังและปีก สีเหลืองทองบริเวณหน้าอกและท้อง ปากใหญ่แข็งแรง ใช้สำหรับกัดเปลือกถั่วหรือผลไม้ได้ดี มีความฉลาดสูงและสามารถฝึกพูดได้


ถิ่นที่อยู่ : พบในป่าฝนเขตร้อนของทวีปอเมริกาใต้ โดยเฉพาะในแถบลุ่มแม่น้ำอเมซอน


การสืบพันธุ์ : นกแก้วมาคอร์ทำรังในโพรงไม้ใหญ่ วางไข่ครั้งละ 2-3 ฟอง ใช้เวลาฟักประมาณ 26-28 วัน โดยพ่อแม่นกช่วยกันดูแลลูกอ่อนอย่างดี
