นกกาน้ำเล็ก

ชื่อวิทยาศาสตร์ : Microcarbo niger
วงศ์ : Phalacrocoracidae
อันดับ : Pelecaniformes

ลักษณะ : มีความยาวจากปลายปากถึงปลายหาง 51-56 เซนติเมตร น้ำหนัก 360-525 กรัม ความกว้างระหว่างปลายปีกทั้งสอง 90 เซนติเมตร ตัวผู้และตัวเมียมีลักษณะเหมือนกัน ขนตามลำตัวมีสีดำเหลือบน้ำเงิน ตรงปีกมีสีน้ำตาลปน บริเวณรอบตามีจุดสีขาวขนาดเล็ก ใต้คางมีสีครีม ปากสีเนื้อแกมเทาหม่น ขนทั่วทั้งหัว, ลำคอ, ลำตัว ปีกและหางสีน้ำตาลแกมดำ แต่ขนบริเวณไหล่และปีกสีค่อนข้างเทา แต่ขอบขนสีดำ คางค่อนข้างขาว ในฤดูผสมพันธุ์ ปากสีค่อนข้างดำ หัว, ลำคอ, อก, ท้อง, สีข้างและขนคลุมใต้โคนหางเปลี่ยนเป็นสีดำเหลือบน้ำเงินและเขียว บนกระหม่อมขนคลุมหู และท้ายทอยมีลายริ้วสีขาว


ถิ่นที่อยู่ : ในประเทศไทยพบทั่วไปทุกภาค อาศัยอยู่ตามหนองบึง, แม่น้ำ, ลำคลอง หรือท้องนา จับปลาขนาดเล็กจำพวกปลาตะเพียนกินเป็นอาหาร ส่วนใหญ่ชอบอยู่ตามลำพัง บางครั้งอาจพบอยู่รวมกันเป็นฝูงบ้าง ชอบดำน้ำไล่จับปลาเป็นอาหาร เมื่อขึ้นจากน้ำมักจะยืนกางปีกตากแดดให้ขนแห้ง


การสืบพันธุ์ : นกกาน้ำเล็กผสมพันธุ์ในราวเดือนกรกฎาคม ทำรังอยู่บนต้นไม้ใหญ่รวมกันหลายรังบนต้นเดียวกัน ทำรังด้วยกิ่งไม้เล็ก ๆ ขัดสานไว้อย่างหยาบ ๆ และวางไข่ครั้งละ 2-4 ฟอง ไข่มีลักษณะฟ้าจางๆ ซึ่งพ่อกับแม่นกจะช่วยกันกกไข่ราว 25 วัน

นกกาแวน

ชื่อวิทยาศาสตร์ : Crypsirina temia
วงศ์ : Corvidae
อันดับ : Passeriformes

ลักษณะ : เป็นนกขนาดเล็กลำยาวประมาณ 33 เซนติเมตร ลำตัวเพรียว มีขนสีดำเหลือบเขียวคลุมทั่วทั้งตัว ม่านตาสีฟ้าอมน้ำเงิน มีขนสีดำคล้ายกำมะหยี่อยู่รอบดวงตาและด้านหน้า จะงอยปากใหญ่สีเทาดำ ปลายปีกมน และหางยาวโดยโคนหางเรียวเล็ก ค่อยๆกว้างขึ้นจนถึงปลาย หางมีลักษณะคล้ายพัด หรือไม้แร็คเก็ต


ถิ่นที่อยู่ : พบได้ตามป่าละเมาะ ป่าเบญจพรรณ รุ่น ป่าชายเลน ป่าไผ่ และทุ่งโล่ง เป็นนกที่ชอบอาบน้ำและรักษาความสะอาดขน มักหลบซ่อนตัว หากินอยู่ในที่รกทึบ บนต้นไม้ พุ่มไม้ หรือตามป่าไผ่ อาหารท่ชกินคือแมลว สัตว์ขนาดเล็ก และผลไม้บางชนิด


การสืบพันธุ์ : ผสมพันธุ์ช่วงเดือนเมษายน-พฤษภาคม ทำรังอยู่บนต้นไม้หรือกอไผ่ รังมีลักษณะเป็นรูปถ้วยทำจากกิ่งไม้และเศษหญ้า แลรองด้วยวัสดุอ่อนนุ่ม เช่นใบไม้หรือดอกหญ้า แล้วจึงวางไข่ ไข่มีลักษณะสีน้ำตาลอ่อน มีจุดสีน้ำตาลเข้มกระจายอยู่โดยรอบ ไข่ครั้งละ 2-4 พอง พ่อและมานกจะช่วยกันกกไข่ประมาณ 14-15 วัน จากนั้นจะคอยดูแลลูกนกไปอีก 20-25 วันจนลูกแข็งแรงและสามารถออกหากินด้วยตัวเองได้

นกกาเขาไฟ

ชื่อวิทยาศาสตร์ : Streptopelia tranquebarica
วงศ์ : Columbidae
อันดับ : Columbiformes

ลักษณะ : เป็นนกตัวไม่โตลำตัวความยาวไม่เกิน 23 เซนติเมตร ดวงตากลมดำ ปากเล็กตัวอ้วนป้อม ห้างสั้น นกตัวผู้มีหัวสีเทาน้ำเงิน ปีกและลำตัวสีน้ำตาลแดง และมีแถบสีดำที่ต้นคอ นกตัวเมียมีลักษณะคล้ายกัน แต่มีขนสีอ่อนกว่ามาก


ถิ่นที่อยู่ : นกชนิดนี้ชอบอยู่เป็นคู่ หรือหากินร่วมกับฝูงเล็กๆ ตามป่าโปร่ง ป่าละเมาะ และท้องไร่ท้องนาทั่วทุกภูมิภาคของไทย ยกเว้นภาคใต้ ตอนกลางวันมักออกหาเมล็ดข้าว ธัญพืช และแมลงขนาเล็กเป็นอาหาร เมื่อพลบค่ำก็นอนบนต้นไม้กับคู่


การสืบพันธุ์ : ผสมพันธุ์ราวเดือนมีนาคม-สิงหาคม จะช่วยกันสร้างรังบนกิ่งไม้สูง 2-6 เมตร เพื่อนกตัวเมียจะเข้าไปวาวไข่สีขาว ครั้งละ 2-3 ฟอง โดยนกตัวผู้จะช่วยกกไข่ในตอนกลางวัน ราว 13-14 วัน ลูกนกจึงเจาะเปลือกออกมา

นกเงือกสีน้ำตาล

ชื่อวิทยาศาสตร์ : Anorrhinus tickelli
วงศ์ : Bucerotidae
อันดับ : Coraciiformes

ลักษณะ : มีโหนกเล็ก จะงอยปากสีเหลืองและยาว มีหนังรอบตาสีฟ้า ลำตัวด้านบนสีน้ำตาล ด้นล่างแกมส้ม ปลายปีกขาว ปลายขนหาง ยกเว้นคู่บนสุดสีขาว ไม่มีขนปกคลุมใต้ปีก ตัวผู้มีคอสีน้ำตาลแกมส้ม ตัวเมียมีปากเทาดำ มีขนาดโตเต็มที่ประมาณ 60-65 เซนติเมตร จัดเป็นนกเงือกขนาดกลาง


ถิ่นที่อยู่ : ในประเทศไทยมีถิ่นอาศัยในพื้นที่ภาคเหนือตอนล่าง เป็นป่าจำพวกป่าดิบ, ป่าโปร่ง และป่าเชิงเขา มักเกาะอยู่บนกกิ่งไม้ และต้นไม้ที่มีผลไม้สุก โดยเฉพาะผลไทร ลูกหว้า เป็นต้น แต่บางครั่งก็กินสัตว์ขนาดเล็กเป็นอาหารด้วย


การสืบพันธุ์ : นกชนิดนี้ผสมพันธุ์ระหว่างเดือนมกราคม-พฤษภาคม วางไข่สีขาว ครั้งละ 3-4 ฟอง หลังการผสมพันธุ์แม่นกจะขังตัวเองอยู่ในโพรงรัง เมื่อกกไข่และดูแลลูกตลอด 3 เดือน โดยใช้ดินปิดปากโปรงไว้ แต่จะเปิดช่องว่างไว้สำหรับให้พ่อนกคอยหาอาหารมาป้อนแม่นกและลูกนก

นกตะขาบทุ่ง

ชื่อวิทยาศาสตร์ : Coracias affinis
วงศ์ : Coraclidae
อันดับ : Coraciiformes

ลักษณะ : นกชนิดนี้เป็นนกขนาดเล็กราว 30-33 เซนติเมตร มีลักษณะขนสีฟ้าและน้ำเงิน คอสีน้ำตาลเข้มและแกมด้วยสีครามเด่นชัด ท้ายทอยและหลังสีน้ำตาลมะกอกเข้ม ปลายหางแต้ทด้วยสีน้ำเงินอมม่วง หรืออาจมีสีน้ำตาลแทรกอยู่ด้วย


ถิ่นที่อยู่ : ในประเทศไทยพบอยู่ทั่วทุกภาค ยกเว้นทางตอนใต้ของภาคใต้ พบได้ที่ความสูงมากว่าหรือประมาณ 1,500 เมตร จากระดับน้ำทะเล พบในป่าดิบชื้น ป่าเบญจพรรณ ป่าไผ่ ป่าฟื้นตัวที่ค่อนข้างทึบ มักพบที่ความสูงต่ำกว่า 900 เมตรจากระดับน้ำทะเล ที่ระดับ 1,500 เมตรพบได้บ้างจนถึงหายาก พบที่เทือกเขาของภาคตะวันออก เทือกเขาที่กั้นภาคกลางตอนบนกับภาคตะวันออกเฉียงเหนือ ภาคเหนือด้านตะวันออก และผืนป่าตะวันตกตอนบน


การสืบพันธุ์ : นกตะขาบทุ่งผสมพันธุ์ในช่วงเดือนมีนาคม-พฤษภาคม ทำรังบนต้นไม้บริเวณโพรงไม้ที่เกิดขึ้นเองตามธรรมชาติในบางครั้งจะแย่งรัง หรือโพรงเก่าของนกอื่นเพื่อวางไข่ ไข่มีลักษณะสีขาวมันวาว ตัวเมียวางไข่ครั้งละ 2-5 ฟอง

นกแต้วแร้วหูยาว

ชื่อวิทยาศาสตร์ : Pitta phayarei
วงศ์ : Pittidea
อันดับ : Passeriformes

ลักษณะ : นกขนาดกลาง ลำตัวป้อม ขาสั้น หางสั้น บินไม่เก่งนัก ความยาวลำตัวประมาณ 23-25 เซนติเมตรสีขนโดยรวมออกน้ำตาลอมแดง มีลายพรางที่กลมกลืนกับพื้นป่า จุดเด่นคือ "หูยาว" ซึ่งก็คือขนยาวยื่นออกจากเหนือหู ขนบริเวณนี้เป็นเส้นบาง ๆ สีขาว-ดำ คล้ายหูของสัตว์เลี้ยงลูกด้วยนม


ถิ่นที่อยู่ : ในไทยมักพบในภาคเหนือ (แม่ฮ่องสอน, เชียงใหม่), ภาคตะวันตก (แก่งกระจาน), ภาคใต้บางส่วน อาศัยใน ป่าดิบชื้น, ป่าดิบเขา, และ ป่าเบญจพรรณ ที่มีความชื้นสูง มักพบในระดับความสูง 300-1,500 เมตรจากระดับน้ำทะเล


การสืบพันธุ์ : มักอยู่ในช่วงฤดูฝน-ปลายฤดูฝน ประมาณเดือนพฤษภาคม-สิงหาคม ชอบทำรังบนพื้นดิน ตามพุ่มไม้หรือกองใบไม้หนา รังมักทำจากใบไม้แห้ง มอส รูปร่างกลมคล้ายโดม มีช่องเปิดด้านหน้า วางไข่ครั้งละ 2-4 ฟอง เปลือกไข่สีขาวนวล

นกปีกลายสก๊อต

ชื่อวิทยาศาสตร์ : Garrulus glandarius
วงศ์ : Corvidea
อันดับ : Passeriformes

ลักษณะ : ลำตัวขนาดกลาง ความยาวประมาณ 33 เซนติเมตร ขนาดใกล้เคียงกับนกกา ขนสีน้ำตาลอมแดงถึงสีน้ำตาลอ่อน ปีกมีลวดลายที่โดดเด่นคือมีริ้วสีฟ้าผสมลายขวางในปีก ตะโพกสีขาว กระหม่อม หาง และส่วนบนบางส่วนมีสีดำ บริเวณหน้าผาก แก้ม คาง และคอเป็นสีขาว


ถิ่นที่อยู่ : ในประเทศไทยพบทางภาคเหนือ ภาคตะวันตก และภาคตะวันออกเฉียงเหนือ ส่วนใหญ่พบในป่าเต็งรัง ป่าดิบแล้ง และป่าเบญจพรรณ ชอบอาศัยอยู่บนยอดไม้ ไม่ค่อยลงมาบนพื้นดินมากนัก มักอยู่ตัวเดียว หรือเป็นคู่ บ้างก็เป็นฝูงเล็ก


การสืบพันธุ์ : ฤดูผสมพันธุ์อยู่ในช่วงเดือนมีนาคม-พฤษภาคม โดยที่รังจะถูกสร้างบนกิ่งไม้ โดยใช้วัสดุธรรมชาติเช่น ไม้ กิ่ง และใบไม้ เพื่อใช้วางไข่ครั้งละ 4-5 ฟอง ไข่มักมีสีพื้นน้ำตาลแกมเขียว พร้อมลวดลายจาง ๆ ช่วงเวลาฟักไข่ประมาณ 17-18 วัน

นกยางกรอกพันธุ์จีน

ชื่อวิทยาศาสตร์ : Ardeola bacchus
วงศ์ : Ardeidea
อันดับ : Ciconiiformes

ลักษณะ : นกที่มีขนาดความยาวลำตัวราว 47 เซนติเมตร เมื่อถึงฤดูผสมพันธุ์ หัว คอ และอกเป็นสีแดงหรือน้ำตาลเข้ม พร้อมมีแผงขนนูนยาวที่ท้ายศีรษะและคอ หลังและปีกส่วนบนมีสีสลับโทนเทาเข้ม หรือฟ้าเข้ม ในขณะที่ปีกด้านในและท้องเป็นสีขาว และเมื่อนอกฤดูผสมพันธุ์ จะมีสีคล้ำโทนเทา น้ำตาล มีจุดหรือริ้วสีอ่อนสลับขาวเล็กน้อย ปากสีเหลือง ปลายปากเป็นสีดำ และโคนปากบนมีสีฟ้าหรือเทาอ่อนในบางตัว ขาและตาเป็นสีเหลือง


ถิ่นที่อยู่ : ในประเทศไทยพบได้ทั่วทุกภูมิภาค มักพบในแหล่งนํ้าจืดและน้ํากร่อย เช่น บึง, หนอง, ทุ่งนาข้าว, คลอง, พื้นที่ชุ่มชื้นทั่วไปที่ระดับน้ำตื้น โดยเฉพาะในแหล่งที่อากาศอุ่น แถวลุ่มนํ้าและพื้นที่ราบลุ่ม


การสืบพันธุ์ : มีฤดูผสมพันธุ์อยู่ในช่วงเดือนเมษายน-สิงหาคม รังมักตั้งอยู่รวมกับนกยางชนิดอื่น โดยที่ไข่จะมีสีเขียวอมฟ้าหรือฟ้าคราม ประมาณ 3-6 ฟอง ใช้ระยะเวลาฟักไข่ราว 18-22 วัน

นกหัวขวานจิ๋วคิ้วขาว

ชื่อวิทยาศาสตร์ : Sasia ochracea
วงศ์ : Picidea
อันดับ : Piciformes

ลักษณะ : เป็นนกหัวขวานขนาดเล็กมาก ตัวเต็มวัยมีความยาวรวมประมาณ 10 เซนติเมตร ขนส่วนหลังสีเขียวมะกอกปนโทนคัสเนต ท้องมีสีซีด หรือสีน้ำตาลแดงบางครั้งอาจมีสีเหลืองอ่อนที่ด้านข้างลำตัว หัวมีมงกุฎ เป็นสีเขียว มีแถบขาวอยู่เหนือตาและต่อจากหลังตา


ถิ่นที่อยู่ : ในประเทศไทยพบในป่าที่มีความชื้นหรือป่าดิบเขา ที่มีไม้ไผ่ เถาพืชพันธุ์ขึ้นเยอะ มักอาศัยในชั้นรกต่ำของป่า ไม่ได้อยู่สูงมากนัก มักไม่อยู่นิ่งชอบกระโดดไปตามกิ่งไม้ หรือต้นไม้เรื่อยๆ บางครั้งจะอยู่รวมกันเป็นคู่ หรืออาจเข้าไปอยู่ในฝูงนกหลายชนิด


การสืบพันธุ์ : ฤดูผสมพันธุ์มักเกิดในช่วง มีนาคม-กรกฎาคม สร้างรังโดยใช้ ไม้ไผ่ นกจะเลือกชิ้นไม้ไผ่แตกหรือชิ้นไม้ไผ่ที่ไม่สมบูรณ์ แล้วหัก แกะ และขยายช่องเพื่อทำรูรังภายในเพื่อใช้วางไข่ครั้งล่ะ 3-5 ฟอง ทั้งพ่อและแม่ช่วยกันฟักไข่ โดยใช้เวลาราว 12-14 วัน

นกเอี้ยงสาริกา

ชื่อวิทยาศาสตร์ : Acridotheres tristis
วงศ์ : Stunidea
อันดับ : Passeriformes

ลักษณะ : ลำตัวขนาดกลาง ยาวประมาณ 23-25 เซนติเมตร เมื่อปีกกางมีแถบสีขาวชัดเจน ขณะบินจะเห็นแถบปีกขาวเด่นออกมา ลำตัวสีน้ำตาลเข้ม-น้ำตาลแก่ แกมเทา ส่วนหัวถึงคอเป็นสีดำ ปาก ขา และแถบหนังเปลือยรอบดวงตาเป็น สีเหลือง ส่วนใต้ลำตัวและหางมีสีอ่อนกว่า บริเวณหางมีปลายสีขาว


ถิ่นที่อยู่ : เป็นนกประจำถิ่นในไทย พบได้ทั่วไปในเขตเมือง ชานเมือง ทุ่งนา พื้นที่เกษตร และบริเวณที่มีมนุษย์อาศัยอยู่ อาศัยตามอาคาร บ้านเรือน โครงสร้างต่าง ๆ เช่น ร่องรอยในกำแพง รอยรังเก่า หรือโพรงไม้ มักหาอาหารบนพื้น เดินได้ดี บางครั้งกระโดดเพื่อเคลื่อนที่


การสืบพันธุ์ : ผสมพันธุ์ได้ตลอดปี ขึ้นอยู่กับพื้นที่และสภาพแวดล้อม รังมักสร้างในโพรงต้นไม้ โพรงกำแพง หรือโครงสร้างของมนุษย์ จำนวนไข่เฉลี่ยในแต่ละรังอยู่ที่ 4-6 ฟอง ระยะเวลาฟักไข่ประมาณ 17-18 วัน
